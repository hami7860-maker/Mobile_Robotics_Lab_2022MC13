1. Node: A process that performs computation.
2. Topic: A named communication channel for streaming messages.
3. Package: A folder with ROS 2 code, dependencies, and build information.
4. Workspace: A folder that contains one or more ROS 2 packages and build outputs.
5. Sourcing: It updates environment variables so the terminal can find ROS 2 and your workspace.
6. colcon build: It compiles the code and generates the build, install, and log folders
